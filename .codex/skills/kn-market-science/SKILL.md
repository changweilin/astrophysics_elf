# Kerr-Newman Market Science

Use this skill when the task is about turning the simulator into a product,
research demo, education asset, or measurable market experiment.

## Evidence Rules

1. Separate facts, assumptions, and recommendations.
2. Use current external sources for market size, competitor claims, pricing,
   curriculum standards, grant programs, or platform policies. Cite sources.
3. Do not invent numeric TAM/SAM/SOM, conversion rates, or willingness-to-pay.
   If data is unavailable, provide a testable estimate range and validation plan.
4. Tie recommendations to actual project affordances: interactive Kerr-Newman
   sandbox, object library, tidal microscope, MHD monitor, full-physics facade,
   units, benchmarks, ray-tracing preparation, and worker/performance layer.
5. Keep analysis falsifiable. Every strategy should name a metric, threshold,
   collection method, and decision rule.

## Analysis Frame

Use the smallest frame that answers the request:

- Education: learner persona, prerequisite math, curriculum fit, concept
  mastery metric, classroom deployment path.
- Research/portfolio: scientific credibility, reproducibility, benchmark
  evidence, demo narrative, GitHub/readme quality.
- Consumer/science media: aha moment, share loop, retention mechanic,
  simulation presets, visual clarity.
- B2B/planetarium/museum: exhibit workflow, facilitator controls, reliability,
  offline mode, licensing or services package.
- Developer/API: facade clarity, examples, worker readiness, numerical trust,
  package documentation.

## Experiment Design

For each proposed experiment, include:

- Hypothesis.
- Target audience.
- Prototype or page required.
- Success metric and minimum threshold.
- Data collection method.
- Cost and effort.
- Stop, iterate, or scale decision.

## Product Constraints

- The current root app is a browser demo using CDN React and Canvas2D.
- The full physics layer is additive and benchmarked; market claims should not
  imply full numerical relativity or self-consistent GRMHD.
- UI copy currently contains corrupt or mixed glyphs in several files; include
  localization cleanup in any launch-readiness assessment.
- Prioritize observable user value over feature volume.

## Handoff Notes

Return concise tables when comparing segments or experiments. End with the next
highest-confidence test, not a broad wishlist.
